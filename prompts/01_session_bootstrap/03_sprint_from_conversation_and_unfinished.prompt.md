# Session Bootstrap > Sprint From Conversation and What Is Unfinished
@workflow: simple_prompt_generator
@doctype: 01_normal_doctype
@bucket: Session Bootstrap

## Task
Go through the conversation and generate a sprint with ✅ / ❌ checkmarks. Mark what is done versus not done, then explicitly state:
- what is NOT finished,
- what the next concrete step is, and
- what the first prompt of the next session should be.
If you are unsure whether something is really done, verify it in the code before marking it complete.

## Context
[comment]
Use this at the end of a substantial session or after recovering from a compressed or deleted chat.
It turns a messy thread into a real handoff.
[/comment]

Be specific. “Some things still need work” is not enough.
