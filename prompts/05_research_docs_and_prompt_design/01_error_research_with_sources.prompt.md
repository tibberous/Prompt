## Task
Search the web for each error one at a time and write a well-researched fix. Prefer official docs, primary sources, or issue trackers when possible. Include citations and make it clear which source supports which part of the fix.

## Context
[comment]
Use this for tricky runtime, framework, API, or platform errors when memory alone is not trustworthy.
[/comment]

Do not bundle five unrelated errors into one vague explanation. Work them one by one.
